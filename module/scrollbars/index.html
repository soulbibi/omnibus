  • Default : -webkit-overflow-scrolling 적용.
  • style00 ~ style12 : -webkit-scrollbar-track, -webkit-scrollbar, -webkit-scrollbar-thumb 속성 적용
  • -webkit-overflow-scrolling 과 혼용해서 사용하지 못함. 따라서 가속도 스크롤링이 적용안됨.
